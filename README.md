## 개발 기간
2022.8.28 ~ 현재(개발 중단)
```
중단 이유)

[과거] 토블은 초기 React(NextJS)와 NestJS를 배우기 시작해서 시작한 프로젝트로 상당한 스파게티 코드들이 존재.
[흥미] 프로젝트를 계속 진행하기보다는 리액트의 장점인 컴포넌트 재활용성을 적극 활용한 새로운 프로젝트를 진행하고 싶음.
[기술] NestJS에서 TypeORM 를 사용하였는데 Prisma 라는 좋은 기술을 알게되어서 사용해보고 싶음.
```
